## ğŸš¦ Getting Started

### Prerequisites
- Node.js (v18+)
- npm or yarn
- Firebase Account (for backend setup)

### Installation
1.  **Clone the repository:**
    ```bash
    git clone https://github.com/mounikakoppada/campus-compass.git
    cd campus-compass/client
    ```

2.  **Install dependencies:**
    ```bash
    npm install
    ```

3.  **Setup Environment Variables:**
    Create a `.env` file in the `client` directory and add your Firebase and API keys.

4.  **Run locally:**
    ```bash
    npm run dev
    ```
